## Customization Guide

### To Change Primary Color
Edit in `src/app/globals.css`:
```css
:root {
  --primary-50: #[color];
  --primary-100: #[color];
  /* ... rest of scale ... */
}
```

### To Adjust Shadows
Update shadow RGBA values:
```css
--shadow-md: 0 4px 12px rgba(13, 107, 102, 0.08); /* Change 13, 107, 102 */
```

### To Modify Animation Speed
Change duration variables:
```css
--duration-normal: 250ms; /* Faster or slower */
--duration-slow: 400ms;
```
